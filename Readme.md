# AI Personalized Tutor | RAG & Multi-LLM Integration

An intelligent tutoring platform using RAG and multi-LLM routing for 9th-10th grade students.

## Features

- **Hybrid Search**: Semantic + BM25 keyword search
- **Multi-LLM Routing**: phi3:mini (simple) + mistral:7b (complex)
- **Adaptive Teaching**: Beginner/Intermediate/Advanced levels
- **Quiz Generation**: Auto-generated review questions
- **Conversation Export**: Save sessions as JSON
